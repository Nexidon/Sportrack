# Інструкція з генерації документації (Dokka)

Для автоматичної генерації HTML-документації з KDoc-коментарів використовується інструмент Dokka.

## Локальна генерація
1. Відкрийте термінал в Android Studio.
2. Виконайте команду: `./gradlew dokkaHtml`
3. Згенерована документація буде знаходитись у папці `app/build/dokka/html/`.
4. Відкрийте файл `index.html` у браузері для перегляду.

*Примітка для здачі:* Щоб додати архів із документацією (вимога на 85%), перейдіть у `app/build/dokka/`, створіть ZIP-архів папки `html` і відправте його викладачу.

## Контроль якості (Linting)
Dokka автоматично перевіряє наявність документації. Якщо KDoc відсутній для публічного методу, компілятор може видавати попередження (налаштовується через `reportUndocumented = true` в `build.gradle`).
